GitHub Personal Access Token

Provide a GitHub PAT to use the GitHub API for faster updates and workflow triggering.

Current mode: Public CDN (slower, no rate limits)
How to create a PAT:
  1. Go to GitHub Settings → Personal access tokens
  2. Generate new token (classic)
  3. Select scopes:
    • workflow - Required for triggering scorecards
    • Or leave empty for read-only (faster API access only)
  4. Copy and paste here
Benefits with PAT:
  • Faster PR status updates (~30s vs 3-5 min)
  • Bypasses CDN cache for real-time data
  • 5,000 requests/hour limit (vs 60 without)
  • Ability to trigger scorecard workflows
Security:
  • Token stored in memory only (lost on page refresh)
  • Never sent to any server except GitHub
  • You can clear it anytime
